Pair

簡介

pair 可以儲存一對不同類型的資料，由 first 與 second 兩個成員組成

特性

  • 支援不同資料類型的組合
  • 可以進行比較（先比first，若相同則比second）

語法

Vector

簡介

vector 是動態陣列，能夠在執行期間根據需求調整大小，並新增或刪除元素

特性

  • 支援隨機存取
  • 在尾端新增/刪除元素效率較高
  • 在中間插入/刪除元素效率較低
  • 當空間不足時，會自動擴展容量

語法

Stack

簡介

stack 是一種後進先出的容器，就像疊盤子一樣，只能從頂端操作元素

特性

  • 無法遍歷
  • 只能從頂端新增或移除元素

語法

Queue

簡介

queue 是一種先進先出的容器，就像排隊一樣，依照進入順序處理元素

特性

  • 無法遍歷
  • 只能從尾端加入元素，從前端移除元素

語法

Dequeue

簡介

deque 提供了在兩端快速插入和刪除的功能，同時支援隨機存取

特性

  • 支援隨機存取
  • 在兩端插入/刪除的效率都很高
  • 記憶體使用比 vector 更有彈性
  • 雖然方便，但由於常數較大，可能導致運行時間較長

語法

Priority Queue

簡介

priority queue 是一種特殊的佇列，會根據元素的優先級自動排序

語法

Set

簡介

set 是一個有序且不含重複元素的集合

特性

  • 自動排序元素
  • 忽略重複元素
  • 操作效率極高
  • 可以用迭代器遍歷

語法

Map

簡介

map 是一種關聯式容器，可將鍵（key）對應到值（value）

特性

  • 鍵值必須唯一
  • 支援隨機存取
  • 自動依鍵值排序

語法

Bitset

簡介

bitset 是用於處理位元運算的容器

特性

  • 每個位元僅使用 1 bit
  • 位元索引從右至左，最右邊索引為 0
  • 可直接進行 AND、OR、XOR 等位元運算
